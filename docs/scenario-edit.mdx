During scenario editing, you can:

1. **Trigger Tab**: Define trigger conditions to granularly activate the scenario only under certain conditions.
2. **Rules Tab**: Define a set of rules evaluated during the scenario to compute a global score.
3. **Decision Tab**: Define two thresholds used to compare the global score, converting it into an outcome.

You can also change the name and description of the scenario at any time.
