You begin editing a new field in the table by clicking the "Create a new field" button on the table menu.

<Frame caption="Entry point to create a new field">
  <img src="/images/docs/c2b0e4b-Capture_decran_2024-05-29_a_11.48.19.png" />
</Frame>

You are invited to choose values for the following inputs:

| Input                  | Values                                | Description                                                                                                                                                                |
| ---------------------- | ------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Name                   | alphanumeric + "`_`"                  | Unique per table. Case sensitive                                                                                                                                           |
| Description            | free text                             | Description for the scenario builders                                                                                                                                      |
| Required               | true / false                          | Do you want to allow ingestion of objects where this value is missing                                                                                                      |
| Type                   | string / boolean / timestamp / number | Select one of the four values                                                                                                                                              |
| Is an enumerated value | true / false                          | Do you want to keep track of distinct values seen (for contextual help in the rule builder). Incompatible with "unique". Only available for string and number type fields. |
| Is unique              | true / false                          | Is the field unique, i.e. can there be only of row (one `object_id` for every distinct value of this field). Only available for string and number type fields.             |
