# Create or update a workflow

1. Navigate to the Workflows section
2. Search for the scenario for which you wish to edit the workflow.
3. Navigate to the detail view

<Frame caption="Create or update a scenario workflow">
  <img src="/images/docs/3f9aa81-image.png" />
</Frame>

<Info>
  Scenarios with existing workflows are identified using this column
</Info>

4. Edit the workflow detail and save

<Frame caption="Edit workflow">
  <img src="/images/docs/a8fd503-image.png" />
</Frame>

The types of workflows currently available are:

| Workflow type                    | Workflow trigger      | Description                                                                                                                                                |
| -------------------------------- | --------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Create new case for decision     | A decision is created | Create a new case every time a decision is created for this scenario, for specific outcomes.                                                               |
| Add decision to an existing case | A decision is created | Try to add the decision to an open case corresponding to the same [pivot value](/docs/pivot-values-1). If none is present, fall back to creating a new cas |

# Delete a workflow

1. Navigate to the Workflows section
2. Search for the scenario for which you wish to delete a workflow
3. Navigate to the detail view

<Frame caption="Go to scenario workflow detail view">
  <img src="/images/docs/4746ea0-image.png" />
</Frame>

4. Click on the `Delete` red button at the bottom

<Frame caption="Delete an existing workflow">
  <img src="/images/docs/946550e-image.png" />
</Frame>
